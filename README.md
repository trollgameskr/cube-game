# cube-game
3x3큐브게임

## 🎮 게임 소개
브라우저에서 즐기는 3x3 큐브 퍼즐 게임입니다. 터치 스와이프로 큐브를 회전시켜 모든 면을 단색으로 맞추세요!

## ✨ 주요 기능
- 🎯 **터치/스와이프 컨트롤**: 각 면을 탭하고 스와이프하여 회전
- 🎲 **자동 섞기**: 게임 시작 시 랜덤으로 큐브 섞기
- 🏆 **승리 감지**: 모든 면이 단색이 되면 자동으로 축하 메시지 표시
- 📊 **이동 횟수 추적**: 몇 번의 이동으로 퍼즐을 풀었는지 확인
- 🔄 **리셋 기능**: 언제든지 새로운 게임 시작
- 💡 **힌트 기능**: 도움이 필요할 때 힌트 확인
- 📱 **반응형 디자인**: 모바일과 데스크톱 모두 지원
- ⚡ **60FPS 최적화**: 부드러운 애니메이션

## 🎮 게임 방법
1. 각 면을 탭하여 선택
2. 선택한 면에서 상하좌우로 스와이프하여 회전
3. 모든 면을 단색으로 맞추면 클리어!

### 면 표기
- **U** (Up/위): 노란색
- **D** (Down/아래): 하얀색
- **F** (Front/앞): 빨간색
- **B** (Back/뒤): 주황색
- **L** (Left/왼쪽): 초록색
- **R** (Right/오른쪽): 파란색
